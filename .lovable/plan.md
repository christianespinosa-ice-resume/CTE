

# Christian T. Espinosa — Online Resume

## Overview
A clean, professional single-page online resume showcasing Christian's experience as a Plant Automation & SCADA Engineer. The design will be modern, print-friendly, and mobile-responsive.

## Page Layout & Sections

1. **Header** — Full name, job title, location, and contact details (email & phone) with a professional, bold layout
2. **Professional Profile** — Summary paragraph highlighting 9 years of experience
3. **Core Expertise** — Grid/tag-style layout of technical skills (AVEVA, SCADA, PLC, protocols, etc.)
4. **Professional Experience** — Two positions with company details, date ranges, bullet points, and key impact highlights:
   - Texas Instruments (2018–2023) — SCADA Developer / INCS
   - ICI Systems Corporation (2013–2018) — Instrument Technician
5. **Education** — BS in Instrumentation and Control Engineering, Rizal Technological University
6. **Technical Tools** — List of software and tools

## Design Style
- Clean, minimal professional design with a dark navy/slate accent color scheme
- Clear section dividers and readable typography
- Skill tags displayed as badges for visual appeal
- Print-friendly layout with a "Download / Print" button
- Fully responsive for mobile and desktop viewing

## Features
- **Print button** — Allows saving as PDF directly from the browser
- **Responsive layout** — Looks great on all screen sizes
